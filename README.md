# Color Picker
### Windows desktop color pipette

- Live color preview
- History of 20 recent colors
- Result in various formats
  - HEX
  - RGB
    - Byte (0-255)
    - Float (0.000-1.000)
  - RGBA
    - Byte (0-255)
    - Float (0.000-1.000)
  - (CMYK)
